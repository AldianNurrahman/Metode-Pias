Metode Segi Empat

Metode Segi Empat, atau Left Rectangle Rule, mendekati integral dengan memecah area di bawah kurva jadi rectangle‐rectangle kecil (pias), lalu menggunakan tinggi fungsi pada **titik kiri** setiap rectangle.

Rumus

I ≈ h · ∑₍ᵢ₌₀₎ⁿ⁻¹ f(a + i·h)
dengan h = (b − a)/n

Langkah

  1. Tentukan a, b, dan n.
  2. Hitung h = (b − a)/n.
  3. Untuk i = 0…n−1: xi = a + i·h → luasᵢ = f(xi)·h
  4. Jumlahkan semua luasᵢ.

Contoh

ixif(xi)luasᵢ

21.50004.25002.1250
32.00006.00003.0000
42.50008.25004.1250

Metode Titik Tengah

Metode Titik Tengah (Midpoint Rule) membagi area jadi rectangle kecil, tapi memakai **titik tengah** tiap interval untuk menghitung tinggi fungsi—biasanya lebih akurat.

Rumus

I ≈ h · ∑₍ᵢ₌₀₎ⁿ⁻¹ f(a + (i + ½)·h)
dengan h = (b − a)/n

Langkah

  1. Tentukan a, b, dan n.
  2. Hitung h = (b − a)/n.
  3. Untuk i = 0…n−1: xi = a + (i + ½)·h → luasᵢ = f(xi)·h
  4. Jumlahkan semua luasᵢ.

Contoh

ixif(xi)luasᵢ
11.25003.56251.7813
21.75005.06252.5313
32.25007.06253.5313
42.75009.56254.7813
Total12.6250
