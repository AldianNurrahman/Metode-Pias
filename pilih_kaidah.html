Pilih Kaidah Integrasi

Metode Segi Empat

  1. Hitung lebar pias: h = (b - a) / n
  2. Gunakan titik kiri tiap pias: x_i = a + i·h
  3. Jumlahkan f(x_i)·h untuk i=0…n-1

Metode Titik Tengah

  1. Hitung lebar pias: h = (b - a) / n
  2. Pakai titik tengah: x_i = a + (i+½)·h
  3. Jumlahkan f(x_i)·h untuk i=0…n-1
