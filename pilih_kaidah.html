Metode ini menggunakan pembagian interval [a, b] menjadi n subinterval dengan panjang sama. Setiap subinterval dianggap sebagai persegi panjang dengan tinggi f(x) di titik awalnya.

Langkah-langkah:

  1. Hitung panjang setiap pias (h): h = (b - a) / n
  2. Tentukan titik xi = a + i * h untuk tiap pias
  3. Hitung f(xi) untuk tiap pias
  4. Kalikan dengan h untuk dapatkan luas tiap pias
  5. Jumlahkan seluruh luas â†’ hasil integral aproksimasi

  2. Tentukan titik tengah xi = a + (i + 0.5) * h
  3. Hitung f(xi) di titik tengah tiap pias
